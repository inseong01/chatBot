# CHATBOT

## 구현할 것
난이도 : 상, 중, 하   
구현 우선순위로 정렬 

### 통합 
- ~~챗봇 디자인 (하)~~
- ~~`vercel` 배포, 실시간 통신 (중)~~
- 문자 상태 : `작성중`, `보냄`, `읽음` (중)
- 메시지 구분 : `클라이언트` `관리자` (중) - 임의 변수로 설정해놓음
- 챗봇 개발 정보 추가 (하)

### 관리자 
- 채팅방 목록 (상)
- 관리자 온라인 여부 (상)
- 챗봇 구현 : 질문, 응답 전달 (상)

### 클라이언트 
- 키보드 입력 제한 : 관리자 오프라인 (중)
- 마우스로 답변 선택 (중)
