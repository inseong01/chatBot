### 1. 이해하기
   - 기능 소개
     : 챗봇 채팅의 주요 기능과 작동 방식에 대한 설명  
        - 메시지
        - 접속 인원
        - 접속 목록
        - 챗봇

   - 실제 사용 예시 
	gif - 수정 번거로움
	코드 샌드박스 - 환경변수 유출 우려
	사용예시 보여주거나 체험하도록 함

     v 1. 메시지 (메시지 수신/발신 및 상태 처리 샘플 코드) - admin, client 두 명의 창
     v 2. 접속 인원 (실시간 접속 인원 표시 샘플 코드) - join, sync, leave 작동 콘솔로그
     v 3. 접속 목록 (접속 중인 사용자 목록을 받아오는 샘플 코드) - admin, client 접속/해제
     v 4. 챗봇 (사용자와 챗봇의 기본 대화 흐름 샘플 코드) - client

   - 지원하는 기능

     2. 접속 인원 표시 - [JavaScript 예시 링크]
     3. 접속 목록 관리 - [JavaScript 예시 링크]
     4. 챗봇 대화 처리 - [JavaScript 예시 링크]

### 2. JavaScript 세부 설명
	- supabase 설정: supabase 참고(테이블 설계, 요청방법)
	- API 요청방법/결과값/처리방법, 설계한 코드 설명
		0. 초기설정
			supabase install
			테이블 구독
		1. 요청
			요청 코드
		2. 응답
			응답값 예시 코드
		3. 예제
			기능 별 코드 활용

   - 메시지 수신/발신/상태
     : 클라이언트-서버 간 메시지 송수신 및 상태 처리 샘플 코드
   - 접속 인원
     : 실시간 접속 인원 수 관리 로직
   - 접속 목록  
     : 관리자 패널에서 접속 사용자 목록을 표시하는 방법
   - 챗봇  
     : 대화형 인터페이스 구현과 사용자 입력 처리

### 3. 보완 사항 (선택)
   - 상호작용 흐름: 메시지 전송 및 응답 과정을 다이어그램으로 설명
   - 에러 처리: 메시지 전송 실패 시 에러 처리 및 상태 코드 설명
   - 보안 고려 사항: 인증/인가 및 데이터 암호화 처리
   - 테스트 및 디버깅 가이드: 기능별 테스트 및 디버깅 방법
